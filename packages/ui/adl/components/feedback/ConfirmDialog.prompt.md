Confirmação proporcional ao risco — destrutiva explícita; irreversível exige digitar para confirmar.

```jsx
<ConfirmDialog open={open} destructive title="Excluir vaga"
  description="A vaga e as 34 candidaturas serão removidas. Esta ação não pode ser desfeita."
  confirmLabel="Excluir vaga" typeToConfirm="EXCLUIR"
  onConfirm={excluir} onCancel={fechar} />
```
