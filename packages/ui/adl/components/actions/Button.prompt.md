Botão ADL — rótulo é verbo na ação ("Publicar vaga"), nunca "OK".

```jsx
<Button icon="plus">Publicar vaga</Button>
<Button variant="secondary">Cancelar</Button>
<Button variant="destructive">Excluir vaga</Button>
<Button variant="subtle" size="compact" icon="download">Exportar</Button>
<Button loading>Salvando…</Button>
```

Variantes: primary · secondary · subtle · destructive. size="compact" (28px) em tabelas.
