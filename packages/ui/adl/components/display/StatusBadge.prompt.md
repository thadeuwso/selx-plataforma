Badge de status — significado fixo por tom, com ponto (cor nunca é o único canal). Nunca decorativo.

```jsx
<StatusBadge tone="success">Aprovado na etapa</StatusBadge>
<StatusBadge tone="warning">Aguardando retorno</StatusBadge>
<StatusBadge tone="danger">Não aprovado nesta etapa</StatusBadge>
```
