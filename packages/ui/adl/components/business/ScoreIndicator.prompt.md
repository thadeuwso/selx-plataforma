Indicador de score com explicação — todo score de IA vem com origem, critérios e confiança; nunca número seco. Clique abre o "porquê".

```jsx
<ScoreIndicator value={87} label="Match" confidence="alta"
  criteria={[{ label: 'Experiência com dados', value: 92 }, { label: 'Inglês', value: 60 }]} />
```
