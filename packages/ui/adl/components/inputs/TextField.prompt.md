Campo de texto — linha única com rótulo, ajuda e erro ("o que houve + o que fazer").

```jsx
<TextField label="Nome da vaga" placeholder="Ex.: Analista de dados" required />
<TextField label="E-mail" type="email" error="E-mail inválido. Verifique o formato." />
<TextField icon="search" placeholder="Buscar candidatos…" />
```
