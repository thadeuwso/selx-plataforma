Grupo de radio — escolha única entre poucas opções visíveis.

```jsx
<Radio label="Modelo de trabalho" defaultValue="hibrido" options={[
  { value: 'presencial', label: 'Presencial' },
  { value: 'hibrido', label: 'Híbrido', help: '2–3 dias no escritório' },
  { value: 'remoto', label: 'Remoto' },
]} />
```
